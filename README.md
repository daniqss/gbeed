# gbeed
DMG Game Boy emulator for embedded devices. This project aims to provide a simple DMG Game Boy emulator that can run both over a graphical session and in a DRM/KMS environment. This will allow us to run in embedded devices that Linux, with a minimal setup. We will use [Raylib](https://www.raylib.com/) for graphics, input and audio.
