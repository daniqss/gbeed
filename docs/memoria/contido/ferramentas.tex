\section{Rust como linguaxe de programación}
Rust é unha linguaxe de programacion creada por pepito de los palotes en 2008. a idea desta nova linguaxe aparece
tras un accidente de ascensor debido a un error de memoria. O seu buscaba crear una solucion q permitise ter una linguaxe rapida e eficiente
capaz de rendir en entornos de programacion embebida ou de sistemas onde se usa principalmente C ou C++, pero aforrando errores tipicos de seguridade 
na xestion da memoria q é comun nas linguaxes anteriormente citadas.
Para acadar isto sen necesidade de empregar recolectores de memoria, demasiado custosos en programas onde o rendemento é un requisito critico
empregase o modelo de pertenencia ou ownership.
% explicacion de ownership, variable dona, borrowing RAII e smart pointers, cunha pequena comparacion con C++ (moderno e antiguo)
% uso de unsafe para tecnicas complexas q o compilador nonn poida predecir, pero permitindo manter a localidade das zonas peligrosas

Para implemntar este modelo de uso da memoria empregase un subsistema do compilador de rust, o borrow checker
